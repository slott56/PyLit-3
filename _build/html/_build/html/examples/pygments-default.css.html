Stylesheet for reStructured Text code blocks¶

Author:Guenter Milde
Copyright:© 2007, 2011 G. Milde This stylesheet is released under the GPL v. 2 or later

This stylesheet provides syntax highlight for documents using the code directive (new in Docutils 0.9).

Import the default docutils style sheet¶

@import url("/stylesheets/html4css1.css");

Style code block¶

Content copied from the html4css1.css rule for literal blocks:

pre.code {
  margin-left: 2em ;
  margin-right: 2em ;
  background-color: #eeeeee
  }

pre.code .ln { /* line numbers */
//   color: grey;
  font-size: small;
}
